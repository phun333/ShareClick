Want a **free, open-source ShareMouse alternative**? ShareClick shares one mouse,
keyboard, clipboard and files across your Mac and Windows PC — no license nag,
encrypted by default, and open source.

| Feature | ShareClick | ShareMouse |
| --- | --- | --- |
| Price | **Free (always)** | Free for 2 personal PCs; paid otherwise |
| Open source | Yes (MIT / Apache) | No |
| Mac & Windows | Yes | Yes |
| Easy auto-setup | mDNS discovery | Yes (auto-detect) |
| Clipboard (text + images) | Yes | Yes |
| File transfer | Yes | Yes (drag & drop) |
| Encryption | X25519 + ChaCha20-Poly1305 | AES |
| Input latency focus | **~6 µs transport (UDP)** | Good |
| Commercial use | **Free** | Requires paid license |

## Which should you pick?

- **Choose ShareClick** if you want it **free forever** (including at work), open
  source, encrypted by default, and as low-latency as possible.
- **Choose ShareMouse** if you want its very polished drag-and-drop and don't mind
  a paid license for more than two computers or commercial use.

ShareMouse is a slick, well-established tool with excellent file drag-and-drop.
ShareClick matches the core feature set — input, clipboard (text + images),
files — while staying free, open source, and license-free for any number of
machines.

shareclick

one keyboard & mouse across your Mac and Windows PC. clipboard and files too. encrypted, open-source, and built for the lowest input lag.

LAN-only · self-hosted

everything, nothing extra

the good parts of ShareMouse, Synergy and Deskflow — free, open, and faster.

~6µs

lowest input lag

UDP input path on a dedicated socket, events coalesced per tick. the network is the only bottleneck — not the code.

end-to-end encrypted

X25519 + a shared passphrase authenticate both machines, then ChaCha20-Poly1305 seals every packet. ~20 ns cost.

clipboard & files

copy text or images on one machine, paste on the other. send files straight across the encrypted channel.

seamless switching

push the cursor into a screen edge and control jumps to the other machine. push back and it returns. no hotkey needed.

zero-config discovery

peers find each other on the LAN over mDNS. no IP addresses to type — the passphrase still proves identity.

open source

MIT / Apache-2.0. a single tiny binary, menu-bar on macOS, system tray on Windows. inspect it, fork it, run it.

up and running in a minute

install on both machines, share a passphrase, done.

01

install

grab the .dmg on your Mac and the .exe on your PC. both launch to the menu-bar / tray.

02

set a passphrase

open Settings & Monitor Manager, pick the same secret passphrase on both, and lay out which screen is where.

03

just move

slide your cursor to the shared edge — your keyboard & mouse now drive the other machine. clipboard syncs automatically.

macOS asks for Accessibility + Input Monitoring permission on first launch†. Windows asks to allow it through the firewall‡. both machines must be on the same network.

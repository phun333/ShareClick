which should you pick?

Choose ShareClick if you want it free forever (including at work), open source, encrypted by default, and as low-latency as possible.

Choose ShareMouse if you want its very polished drag-and-drop and don't mind a paid license for more than two computers or commercial use.

ShareMouse is a slick, well-established tool with excellent file drag-and-drop. ShareClick matches the core feature set — input, clipboard (text + images), files — while staying free, open source, and license-free for any number of machines.
